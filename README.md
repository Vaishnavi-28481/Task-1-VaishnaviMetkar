# Project 1 - Data Cleaning & Preparation

## Overview

This project focuses on cleaning and preparing raw business transaction data for further analysis.

The dataset was processed to improve data quality and consistency before performing analysis and SQL operations.

---

# Tasks Performed

* Handled missing values
* Checked duplicate records
* Standardized column names
* Corrected data types
* Prepared clean dataset for analysis

---

# Key Findings

* Dataset contains 1200 rows and 14 columns
* Missing values found only in `coupon_code`
* 309 missing coupon values identified
* No duplicate records found
* Column names converted to lowercase format
* Cleaned dataset saved successfully
